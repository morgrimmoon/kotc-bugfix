# kotc-bugfix
King of the Castle Unofficial Bugfix Mod v2

I have no connection to Team17 or Tributary games! This is 100% a fan project. Anything I break is on me and not on them; bugs you encounter while using this mod should be reported to here first. Bugs you encounter while playing the unmodded game should also be reported here so I can do something about it.

## Submitting Bugs
Make a new issue. Please include:
- which event has the issue
- whether it was a party or twitch game
- for anything more complex than a typo, a screenshot or twitch clip is very helpful
- if you're willing to chat with me on the Tributary Discord, if it's a more complicated issue

## Installation Instructions
### Alpha Testers
Unzip the file in a suitable location, and drag the .txt files into your pre-configured events folder.

### Everyone Else
Don't install from here. There'll be a NexusMods download once it's ready for prime time. If you want to join in the testing, poke me in the code-wizards channel.
